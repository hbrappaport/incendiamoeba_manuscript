Analysis code for biophysical properties 

Content includes: 

  1_analyze_pdb_structures: PDB files of highly transcribed sequences predicted via Alphafold2 and script to extract biophysical properties 
  
  2_plot_parameters: plotting scripts for Fig. 4F and Fig. S6 
  
  3_APBS: aligned PDB files for dihydropteridine reductase, script to perform APBS calculation, and APBS output
  
  Tm predictions for I. cascadensis and V. vermiformis in allTms.csv
